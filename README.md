# new-years-revolution

**New Years Revolution** est une application de collecte et de suivi des bonnes résolutions.

Développé par <u>Rémi Matrod</u> et <u>Virgile Jacquet</u>

## Fonctionnement
L'application peut être lancée via un terminal :  
`mvn spring-boot:run`

 - Page d'accueil : http://localhost:8080/
